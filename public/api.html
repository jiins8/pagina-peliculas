Documentacion de la API

/api/peliculas GET
Devuelve una lista de todas las películas en la base de datos
/api/actores GET
Devuelve una lista de todas los actores en la base de datos
/api/peliculas POST
Agrega una nueva película a la base de datos
/api/actores POST
Agrega un/a nuevo/a actor/iz a la base de datos
/api/peliculas/:id GET
Devuelve una película específica según su ID
/api/actores/:id GET
Devuelve un/a actor/iz específico/a según su ID
/api/peliculas/:id POST
Actualiza una película específica según su ID
/api/actores/:id POST
Actualiza un/a actor/iz específico/a según su ID
/api/peliculas/:id DELETE:
Elimina una película específica según su ID
/api/actores/:id DELETE:
Elimina un/a actor/iz específica según su ID
